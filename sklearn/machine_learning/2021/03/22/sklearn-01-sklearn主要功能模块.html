对各模块简要说明

模型相关模块 - 机器学习各种算法模型、函数：

  • calibration
    分类模型预测概率校准模块，对于某些比较强调预测概率准确性的模型，使用该模块对模型的预测概率进行校准。
  • cluster
    用于非监督聚类的模块，包含聚类相关的各种算法模型、函数。
  • covariance
    协方差评估模块
  • cross_decomposition
    交叉分解模块
  • datasets
    数据集模块，除了一些著名的真实数据集的函数外，还包含一些人造数据集函数。
  • decomposition
    矩阵分解模块，主要用于数据降维。
  • dummy
    虚拟模型模块，快速创建虚拟分类或回归模型，用来作基准模型等。
  • ensemble
    集成学习模块，包含各种基于集成学习的分类、回归和异常检查模型。
  • experimental
    试验性控制模块，用于控制实验性接口的行为。
  • feature_extraction
    特征提取模块，提供了常用的文本和图片特征提取工具。
  • feature_selection
    特征选择模块，提供了各种特征过滤、特征消除方法。
  • gaussian_process
    高斯过程模块，实现了高斯过程回归和高斯过程分类。
  • inspection
    查看模型模块
  • isotonic
    保序回归模块
  • kernel_approximation
    核逼近模块
  • kernel_ridge
    核岭回归模块
  • linear_model
    线性模型模块，包含了大量的线性模型。
  • manifold
    流形学习模块
  • metrics
    模型评估模块
  • mixture
    混合模型模块，包括高斯混合模型和贝叶斯高斯混合模型。
  • model_selection
    模型选择模块
  • multiclass
    多分类模块
  • multioutput
    多输出模块
  • naive_bayes
    朴素贝叶斯算法模块，实现了伯努利贝叶斯、高斯贝叶斯、多项式贝叶斯、补充朴素贝叶斯、类别贝叶斯。
  • neighbors
    k近邻算法模块，包含各种常见的k近邻分类、回归模型。
  • neural_network
    神经网络模块，实现了伯努利受限玻尔兹曼机、多层感知机分类模型和多层感知机回归模型。
  • pipeline
    管道模块，用来创建机器学习管道。
  • preprocessing
    数据预处理模块，包含各种数据缩放、中心化、标准化、二值化方法。
  • random_projection
    随机映射模块
  • semi_supervised
    半监督学习模块
  • svm
    支持向量机模块
  • tree
    基于树模型的模块
  • discriminant_analysis
    判别分析模块
  • impute
    缺失值处理模块
  • compose
    用来构建复合转换器模型的模块

sklearn框架工程实现相关：

  • exceptions
    sklearn中的告警、报错模块，框架工程辅助模块，非模型、算法相关。
  • externals
    外部接口模块，提供与sklearn兼容的非sklearn模块。
  • clone
    用来克隆带有相同模型参数但未经训练的评估器
  • get_config
    获得环境配置的函数
  • set_config
    设置环境配置的函数
  • config_context
    用来配置sklearn全局环境的管理器
  • show_versions
    用于展示sklearn框架系统信息以及依赖信息的函数
